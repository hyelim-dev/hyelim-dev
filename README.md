## 안녕하세요, 프론트엔드 개발자 혜림입니다 👋

사용자 흐름과 데이터 구조를 함께 설계하며  
단순 UI 구현을 넘어 서비스 구조를 고민하는 개발자입니다.

어드민 시스템과 대시보드 UI처럼  
복잡한 데이터와 상태를 다루는 문제를 해결하는 것을 좋아합니다.

---

### 🧭 소개
- 프론트엔드 개발 경력 5년
- React / Next.js / TypeScript 기반 실무 경험
- 도메인 간 데이터 흐름을 구조화하고  
  상태 중심 UI를 설계하는 데 강점을 가지고 있습니다.
- 실무 시스템을 기반으로 한 포트폴리오 구축 경험 (HMS Admin)

---

### 🛠 기술 스택
- **언어**: TypeScript, JavaScript (ES6+)
- **프레임워크**: React, Next.js, Vue
- **상태 관리**: Recoil, Redux
- **스타일링**: Tailwind CSS, styled-components
- **데이터**: GraphQL (Apollo Client)
- **기타**: MSW, React Hook Form, Framer Motion
- **도구**: Git, Figma

---

### 🧩 관심 분야
- 상태 기반 UI 설계 (State-driven UI)
- 어드민 시스템 / 대시보드 구조 설계
- 데이터 흐름 중심의 프론트엔드 아키텍처
- 성능 최적화 및 렌더링 구조 개선
